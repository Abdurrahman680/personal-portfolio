# CV File Instructions

## To complete the CV feature setup:

1. **Create your CV PDF file** using your preferred tool (Microsoft Word, Google Docs, Canva, etc.)

2. **Export it as a PDF** with the filename: `Muhammad_Abdurrahman_CV.pdf`

3. **Place the PDF file** in the `public` folder of your project:
   ```
   e:\My_Portfolio\public\Muhammad_Abdurrahman_CV.pdf
   ```

4. The CV button is already configured to:
   - Open the PDF in a new browser tab
   - Allow users to view and download
   - Display with a professional download icon
   - Show smooth hover animations

## Current Button Features:
✅ Professional gradient design (purple to pink)
✅ Animated download icon (bouncing effect)
✅ Opens in new tab
✅ Smooth hover effects
✅ Fully responsive (mobile & desktop)
✅ Side-by-side with Education button on desktop
✅ Stacked on mobile devices

Once you place your CV PDF in the public folder, the button will work perfectly!
